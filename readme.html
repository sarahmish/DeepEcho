Overview¶

The DeepEcho library implements methods for modeling mixed-type multivariate time series. It provides multiple models and benchmarks to enable rapid research and development.

Installation¶

To install the DeepEcho library from source, run the following:

git clone https://github.com/sdv-dev/DeepEcho
cd DeepEcho
make install-develop
